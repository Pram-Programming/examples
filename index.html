Hello, World!

영덕중학교 프로그래밍 동아리 PRAM 웹페이지입니다. 아래 링크를 클릭하면 예제 페이지로 이동됩니다.
  • Hello Alert (버튼을 누르면 Hello 알림)
  • Counter (버튼을 누르면 숫자가 증가하거나 감소함)
  • Simple Calculator (숫자를 입력하고 버튼을 누르면 계산 결과가 표시됨)
  • Random Circle Generator (랜덤한 위치와 사이즈로 원이 그려짐)
  • Simple Drawing Tool (마우스를 누르는 동안 원이 생성되는 간단한 그림판)
  • WASD Movable Circle (WASD 키로 원을 움직임)
  • Obstacle Game (장애물을 피하는 게임)
  • Smooth Up-and-down Obstacle (부드럽게 왕복으로 움직이는 장애물)
  • Orbiting Obstacle (원운동하는 장애물)
